## Übung (Optionen)
### Aufgabenstellung
Analysiere die Bedeutung der folgenden Optionen:

$ nmap -Pn -sS -sV -sC -oN nmap_default.txt 10.10.92.153
Was führt nmap in diesem Fall für einen Scan aus? Wie viele Ports werden gescannt. Scanne mit diesem Befehl die metasploitable VM.

### Aufgabe

| Optionen | Bedeutung |
|----------|----------|
| -Pn   |  Host-Discovery ausschalten. Nmap pingt das Ziel nicht an, sondern scannt direkt die Ports  |
| -sS  |   Stealth Scan (SYN-Scan). Schnell, unauffällig, Standard bei vielen Penetrationstests | 
| -sV   |  Versionsscan: Versucht die Versionen der gefundenen Dienste zu identifizieren  | 
| -sC   | 	Führt die Standard-Nmap-Skripte (NSE Default Scripts) aus. Liefert zusätzliche Infos zu Services, Schwachstellen oder Fehlkonfigurationen  |
| -oN nmap_default.txt   | Speichert das Ergebnis im normalen Textformat in die Datei nmap_default.txt  |
| 10.10.92.153   |  Ziel-IP Metasploitable-VM  |

#### Welcher Scan wird ausgeführt?
Der Befehl führt einen SYN-Stealth-Scan mit Service- und Script-Erkennung aus, erweiterten Standard-Scan.

#### Wie viele Ports werden gescannt?
Die 1000 Default Ports.

#### Metasploitable VM Scan
1. Metasploitable pingen
2. nmap -Pn -sS -sV -sC -oN nmap_default.txt 192.168.10.52
Datei mit Scan: "nmap_default.txt"

## Übung (vulnerability scan)
### Aufgabenstellung 1
Aufgabe 1: führe eine Auswahl besonders nützlicher Skripte (Option -sC) beschränkt auf die 100 häufigsten Ports (Option -F) aus.

Anmerkung: nmap kann wertvolle Hinweise auf Schwachstellen liefern, ein vollwertiger vulnerability scaner ist es allerdings nicht, dafür gibt es eigene darauf spezialisierte Tools (z.B.: gvm … Greenbone Vulnerability Manager).
### Aufgabe 
Befehl: nmap -sC -F 192.168.10.52

#### Erklärung:

-sC: Diese Option aktiviert die Skripte von Nmap, die als besonders nützlich erachtet werden, um Informationen wie Versionen von Diensten oder bekannte Schwachstellen zu identifizieren.

-F: Diese Option beschränkt den Scan auf die 100 häufigsten Ports. Das bedeutet, Nmap wird nur die wichtigsten Ports ansteuern, um schneller erste Ergebnisse zu liefern.

10.10.92.153: Die IP-Adresse der Metasploitable VM, die du scannen möchtest.
